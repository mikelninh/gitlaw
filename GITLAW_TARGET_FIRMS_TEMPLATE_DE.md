# GitLaw Target Firms Template

## Fields per firm

- Kanzlei
- Ort
- Website
- Praxisbereich
- Teamgroesse (geschaetzt)
- multilingualer Fit?
- dokumentenintensiver Fit?
- warum GitLaw passen koennte
- Status: A / B / C
- letzter Kontakt
- naechster Schritt

## Quick scoring

### 3-point fit score

1 point each for:

- small firm
- document-heavy workflow
- multilingual or chaotic intake likelihood

### Interpretation

- `3/3` = sofort anschreiben
- `2/3` = gute Reserve
- `1/3` = nur wenn starke persoenliche Verbindung
